# [693. 交替位二进制数](https://leetcode-cn.com/problems/binary-number-with-alternating-bits/)

**难度:** 简单

给定一个正整数，检查他是否为交替位二进制数：换句话说，就是他的二进制数相邻的两个位数永不相等。

 **示例 1:** 

```
输入: 5
输出: True
解释:
5的二进制数是: 101
```

 **示例 2:** 

```
输入: 7
输出: False
解释:
7的二进制数是: 111
```

 **示例 3:** 

```
输入: 11
输出: False
解释:
11的二进制数是: 1011
```

 **示例 4:** 

```
输入: 10
输出: True
解释:
10的二进制数是: 1010
```

## Solution


**Language:** C++
```C++
class Solution {
   public:
    bool hasAlternatingBits(int n) {
        int r = n % 2;
        while (n) {
            if (n % 2 != r) return false;
            n /= 2, r = 1 - r;
        }
        return true;
    }
};
```



```c++
class Solution {
   public:
    bool hasAlternatingBits(int n) {
        n = (n ^ (n >> 1));  //这里还不能向左位移，因为最高位无法被弹出
        return (n & ((long)n + 1)) == 0;
    }
};
```
